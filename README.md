# GC Automatización / GC Automation

Landing page de **GC Automatización** — automatización con IA y software a la medida para PYMEs y fábricas en Colombia.

🌐 **Live:** https://gonza10co.github.io/gc-automation/

## Características

- 🇪🇸/🇺🇸 Bilingüe (ES/EN) con detección automática del idioma del navegador y persistencia
- 🌙 Modo oscuro con toggle, detección del sistema y persistencia
- ✨ Animaciones de scroll (reveals escalonados, gráficas que se dibujan, chat animado)
- 📱 Responsive, accesible y sin dependencias — un solo archivo HTML
- 🔍 SEO: Open Graph, JSON-LD (ProfessionalService), meta bilingüe

## Stack

HTML + CSS + JavaScript vanilla. Sin frameworks, sin build, sin dependencias.

## Desarrollo local

```bash
python -m http.server 8000
# → http://localhost:8000
```
